# Suggested Commands

## Development
- **Run tests**: `pytest`
- **Linting**: `ruff check .`
- **Formatting**: `ruff format .`
- **Run project**: `python -m src.main`

## Git
- **Status**: `git status`
- **Commit**: `git commit -m "message"`
- **Push**: `git push`

## System
- **List files**: `ls`
- **Change directory**: `cd`
- **Search files**: `grep -r "pattern" .`
